# Agent Skills Finder - 简历项目描述

---

## 项目简介

**Agent Skills Finder** 是一个发现和展示 GitHub 上优质 AI Agent 提示词和工作流的平台，帮助开发者快速找到可复用的 Agent 技能。

**项目地址**：https://marslander.cloud

---

## 技术栈

| 层级 | 技术 |
|-----|------|
| 前端 | Vue3 + Vite + Axios |
| 后端 | Spring Boot 3.2 + Java 17 |
| 数据库 | PostgreSQL 15 + Redis 7 |
| 部署 | Docker + Nginx (1Panel) |

---

## 核心功能

1. **GitHub 数据爬取** - 定时从 GitHub 搜索优质 Agent Skills
2. **智能翻译** - 将英文描述翻译为中文
3. **4维度热榜系统** - 总榜 / 飙升榜 / 新秀榜 / 语言榜
4. **全文搜索** - 支持关键词和语言筛选
5. **详情展示** - 查看项目 README 和详细信息

## 技术亮点

### 1. 热榜系统设计
- 采用**定时任务 + Redis 缓存**方案，无需复杂实时计算
- 4个维度满足不同用户需求：
  - **总榜**：Star 总数排名
  - **飙升**：Star 增长最快
  - **新秀**：7天内新建/更新的热门项目
  - **语言**：按编程语言分类 TOP10
- 节省服务器资源，适合小服务器部署

### 2. GitHub 数据爬虫
- 使用 GitHub REST API 获取数据
- 配合 Personal Access Token 避免限流
- 每次爬取记录历史 Star 数，支持趋势计算
- 定时任务每 2 小时自动更新

### 3. 翻译服务集成
- 集成 MyMemory 免费翻译 API
- 支持中英文双语展示
- 智能处理 API 限流和错误

### 4. 数据库设计
- 使用 JPA + Hibernate 自动建表
- 支持字段动态扩展
- 合理设计索引，支持多维度排序查询

### 5. 容器化部署
- 使用 Docker Compose 一键部署
- 前端：Nginx 托管静态资源 + API 反向代理
- 后端：Spring Boot JAR 打包
- 1Panel 可视化服务器管理

## 面试话术

> "这是一个个人项目，主要解决开发者找不到优质 Agent Skills 的痛点。我一个人完成了前后端开发、数据库设计、爬虫实现和部署上线。考虑到服务器资源有限，我设计了轻量级的热榜系统，用定时任务+缓存的方式实现，既保证了性能又节省了成本。"

> "在实现热榜时，我设计了一个简单的逻辑：每次爬取数据时记录历史 Star 数，然后按增长量排序计算出飙升榜。这样不需要复杂的后台计算，前端直接展示排序结果。"

> "部署方面使用了 Docker Compose，通过 1Panel 管理服务器，实现了前后端分离部署，前端 Nginx 负责静态资源和 API 反向代理。"
